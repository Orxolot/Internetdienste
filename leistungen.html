Unsere Leistungen

Wir sind ein führendes Unternehmen, das Ihnen umfassende Lösungen für Ihre digitale Arbeitsumgebung bietet. Unser Fokus liegt auf der Bereitstellung erstklassiger Dienste basierend auf den renommierten Open-Source-Plattformen Nextcloud, Bitwarden und OpenProject.

Nextcloud

Nextcloud ist eine freie Software für das Speichern von Daten auf einem Server. Auf die Daten kann der Anwender sowohl über eine Weboberfläche als auch mit Client-Applikationen zugreifen.

Bitwarden

Bitwarden ist ein Preemium-Open-Source-Passwortverwaltungsdienst, der vertrauliche Informationen wie Website-Anmeldeinformationen in einem verschlüsselten Tresor speichert.

OpenProject

OpenProject ist eine kollaborative Projektmanagement-Software. Die Anwendung wird als kostenfreie "Community Edition" und als kostenpflichtige Enterprise Edition angeboten.
